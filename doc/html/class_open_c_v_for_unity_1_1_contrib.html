Mat  dst,

static

Applies a GNU Octave/MATLAB equivalent colormap on a given image.

Currently the following GNU Octave/MATLAB equivalent colormaps are implemented: enum

// C++ code:

COLORMAP_AUTUMN = 0,

COLORMAP_BONE = 1,

COLORMAP_JET = 2,

COLORMAP_WINTER = 3,

COLORMAP_RAINBOW = 4,

COLORMAP_OCEAN = 5,

COLORMAP_SUMMER = 6,

COLORMAP_SPRING = 7,

COLORMAP_COOL = 8,

COLORMAP_HSV = 9,

COLORMAP_PINK = 10,

COLORMAP_HOT = 11

Parameters
srcThe source image, grayscale or colored does not matter.
dstThe result is the colormapped source image. Note: "Mat.create" is called on dst.
colormapThe colormap to apply, see the list of available colormaps below.
